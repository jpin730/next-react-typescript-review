# Next React TypeScript Review

This is a simple Next.js project that reviews how to use React with TypeScript. It includes a basic setup for a Next.js application, along with some example components and pages. It was created with Node.js 24 and pnpm 10.

## Running the Project

To run the project in development mode, use the following command:

```bash
pnpm dev
```

## Linting and Formatting

On commit, staged files will be automatically formatted with Prettier and linted with ESLint. If there are any issues, the commit will be aborted until they are resolved.
