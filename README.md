# Next React TypeScript Review

This is a simple Next.js project that reviews how to use React with TypeScript. It includes a basic setup for a Next.js application, along with some example components and pages.

A Next.js 16 project exploring React 19 with TypeScript. Built with Node.js 24 and pnpm 10.

## Getting Started

1. **Use the correct Node.js version** via [fnm](https://github.com/Schniz/fnm) (recommended) or [nvm](https://github.com/nvm-sh/nvm):

   ```bash
   fnm use   # or: nvm use
   ```

2. **Install dependencies:**

   ```bash
   pnpm install
   ```

3. **Start the development server:**

   ```bash
   pnpm dev
   ```
